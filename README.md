## 📥 下载使用

到本仓库 **[Releases](../../releases)** 页面下载最新的 `SmartCleaner.exe`，**双击即用**，无需安装、无需配 Python。

- Windows 10 / 11 64 位
- 启动时会弹 UAC 请求管理员权限（用于访问受保护目录），点【是】
- 首次启动有 10 秒强制说明弹窗 + AI Key 配置向导

> ⚠️ 杀毒软件偶尔会误报（PyInstaller 打包通病），加白名单即可。

## ✨ 它能干什么？

打开 Windows 资源管理器，你只能看到一堆文件夹的名字。
SmartCleaner 借助大语言模型 **让你能看懂每个文件夹是干什么的、能不能删、删了会怎么样**，然后你勾选要清理或迁移的项，程序帮你执行。

| 能力 | 说明 |
|---|---|
| 🤖 **AI 决策卡片** | 每个候选都附带"是什么 / 怎么来的 / 删了会怎样 / 怎么恢复" |
| 🌳 **树状扫描结果** | 层级展开，每层显示大小（≥1G 用 GB，否则 MB） |
| 💬 **AI 助手「小净」** | 聊天式操作，会自己调查并给方案，确认后才动手 |
| 🚚 **C 盘无感迁移** | 真实搬到其他盘 + Junction 符号联接，软件感知不到搬家 |
| 🛟 **30 天后悔药** | 所有删除走隔离区，随时一键还原；迁移可一键搬回 |
| 🔍 **自定义扫描** | 多盘同时扫，也能只扫指定文件夹 |
| 🔐 **本地隐私优先** | API Key DPAPI 加密，AI 调用只发文件元信息 |

## 🔑 第一次使用：配置 AI Key

没 Key 只能用本地规则引擎，AI 功能用不了。**推荐 DeepSeek**（国内可直连 + 便宜 + 新用户有免费额度）：

1. 打开 [https://platform.deepseek.com/](https://platform.deepseek.com/) 注册
2. 左侧菜单 **【API Keys】→【创建 API Key】**，复制弹出的 `sk-xxxxxxxx`
3. 回到 SmartCleaner →【⚙️ 设置】→ 粘贴 Key → 点【🔌 测试】→ 点【💾 保存】

也支持 OpenAI、Claude、任何 OpenAI 兼容协议（智谱 / 月之暗面 / 通义等）。

> 💰 一次扫描通常花费 < 0.1 元，10 元能用大半年。
> 🆓 完全不想花钱？设置里勾【离线模式】，只用本地规则引擎。

## 🛡️ 安全保障

下列规则在程序内部**强制执行**，AI 也无法绕过：

- 🔒 **保护清单**：`C:\Windows` / `Program Files` / `System32` 等代码层锁死
- 🛟 **删除走隔离区**：所有清理项进 `<盘符>:\.SmartCleaner_Trash\`，30 天可还原
- ⚠️ **大文件二次确认**：>1GB 的项即使 AI 说安全也会弹窗再问
- 🔐 **API Key 本地加密**：用 Windows DPAPI 绑定当前账户，硬盘被拷走也解不开
- 🚫 **危险操作必须按按钮**：AI 调用清理/迁移工具前一定要求用户在 UI 上点确认
- 🔗 **跳过系统兼容联接**：`My Music` / `My Pictures` 等 XP 遗留联接自动跳过

## ❓ 常见问题

**Q：杀毒软件报毒怎么办？**
A：PyInstaller 打包的 Python 程序经常被误判（启发式扫描通病）。这是开源项目，源码公开，加白名单即可。

**Q：删错文件了能恢复吗？**
A：能。所有删除都进【🛟 还原中心】的隔离区，30 天内一键还原。

**Q：迁移会影响软件使用吗？**
A：不会。SmartCleaner 用 **Junction 符号联接**，原路径仍然可访问，软件感知不到搬家。如果不放心，可以随时点【↩️ 搬回原位】。

**Q：会上传我的文件吗？**
A：**绝不会**。所有文件读写都在本地。调用 AI 时只发送文件**元信息**（路径、大小、扩展名分布等几个数字），不发送任何文件内容。

**Q：用户数据存哪里？**
A：`%APPDATA%\SmartCleaner\`，包括加密的 Key、配置、扫描索引、日志。卸载时手动删这个目录即可彻底清除。

**Q：报 bug 去哪里？**
A：点本仓库 **[Issues](../../issues)** 新开一个，附操作步骤 + 截图 + `%APPDATA%\SmartCleaner\logs\tech.log`。
